## Getting Started

### Prerequisites

- Node v18.18.0 or higher

### Installation

#### Clone the repo

```shell
git clone <repo-url>
cd <repo-name>
```

#### Install Dependencies

```shell
cd jupiter-frontend
cd web
npm install
```

#### Start the web app
(in the web folder)
```
npm run dev
```

## Apps

### web

To access the swap implementation go to localhost:3000/swap

#### Commands

Start the web app

```shell
npm run dev
```

Build the web app

```shell
npm run build
```
